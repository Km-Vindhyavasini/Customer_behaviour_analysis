# Customer_behaviour_analysis
Customer behavior analytics project utilizing Python, SQL, and Power BI to uncover insights and drive data‑driven decisions.

🧠 Overview
This project demonstrates end‑to‑end data analytics — from loading and cleaning datasets in Python to performing exploratory_data_analysis (EDA), executing SQL_queries on PostgreSQL/MySQL/SQL Server, and visualizing insights through Power_BI_dashboards. It concludes with a professional report and presentation built using Gamma.

📊 Dataset
The dataset contains customer‑related attributes used to analyze behavioral patterns and trends. It is loaded and processed using pandas and NumPy for efficient data handling.

🧰 Tools
Tool	Purpose
Python	Data loading, cleaning, and EDA
SQL	Querying structured data
Power BI	Dashboard creation and visualization
Gamma	Report and presentation design


⚙️ Steps
Load_dataset using Python libraries.

Perform_EDA to identify patterns and outliers.

Clean_data and handle missing values.

Run_SQL_queries on PostgreSQL/MySQL/SQL Server.

Build_Power_BI_dashboard for visual insights.

Create_report and PPT_in_Gamma summarizing findings.

📈 Dashboard
The Power BI dashboard highlights key metrics such as customer segmentation, purchase frequency, and churn prediction. It provides interactive visuals for quick decision‑making.

🧾 Results
Improved understanding of customer behavior.

Identified actionable insights for business growth.

Delivered a clear, data‑driven presentation.

🚀 How to Run
Clone the repository.

Install dependencies from requirements.txt.

Run the Python scripts sequentially.

Open the Power BI file to view dashboards.

Review the Gamma presentation for final insights.
